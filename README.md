# C template

For VSCode-based editors and Clang. Requires CodeLLDB extension. clangd extension is recommended.

An assignment for the DTU course Basic C Programming.

The program loads the entire text of Mary Shelley's Frankenstein into a character, cleans it up by replacing non-printable characters with whitespace, reads tokens (words) delimited by whitespace and keep track of which token succeeds which token, and then generates and prints random sentences using the tokens. The output sentences start with a capital letter and end with a '!' or '?'.

Example outputs:
1:
    Come, Victor; not yet have been built for happiness, but as if you desponding and talent who was unwilling to discover so dear Victor, be at least interest in one change of a noisome dungeon while my brethren, my passions and you French?

    Begone!

2:
    Montanvert was of this feeling?

    Finding the idea that this appear!
